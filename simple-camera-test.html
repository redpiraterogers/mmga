📷 Simple Camera Test

Click button below to test camera
What should happen:
1. Click "START CAMERA"
2. Browser asks for camera permission
3. Tap "Allow"
4. Your face appears in the video box above

If it doesn't work:
- Check if you're in Safari/Chrome (not Phantom)
- Check if site is HTTPS (lock icon)
- Check if another app is using camera
- Try closing all apps and refreshing
